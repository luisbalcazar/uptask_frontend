## UpTask (In Development)

Web application for project managing, allowing users to create, edit, and manage projects and adding tasks.

## Tech Stack

- Frontend: React / TypeScript / Vite / TailwindCSS / React Query / React Hook Form / Zod

## Features

- Form management using React Hook Form
- State management with React Query
- User notifications using React Toastify
- Responsive UI built with TailwindCSS

## Project Structure

```
/src → Application source code (components, layouts)
/store → Global state management (Zustand)
/types → Different customed object types
```

## Installation & Setup

```bash
# Clone the repository
git clone https://github.com/luisbalcazar/uptask_frontend.git

# Install dependencies
npm install

# Run project
npm run dev
```

## Technical Highlights

- Implementation of form management using React Hook Form
- State handling using React Query
- User feedback system using React Toastify
- Modular frontend architecture with Vite
